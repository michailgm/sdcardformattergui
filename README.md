# SD Card Formatter GUI (Linux)

Приложение с графичен интерфейс (GUI) за сигурно и лесно форматиране на SD карти под Linux с Tuxera SD Card Formatter ако не е наличен използва системните инструменти.

## 📋 Описание
Това приложение предоставя интуитивен интерфейс за работа с инструмента `sdcardformatter`. Поддържа автоматично засичане на карти, избор на етикет на дяла и различни методи за форматиране.

## ✨ Функционалности
- **Автоматично засичане:** Разпознава в движение поставени SD карти.
- **Опции за формат:** Поддържа "Бърз формат" (Quick) и "Пълен формат" (Overwrite).
- **Многоезичност:** Пълна поддръжка на Български и Английски език (избира се автоматично според системните локали).
- **Защита:** Предпазва от случайно форматиране на системни дискове.
- **Интеграция:** Работи отлично с KDE (KDialog) и GNOME (Zenity) за системни известия.

## 📋 Описание  
Это приложение предоставляет интуитивный интерфейс для работы с инструментом `sdcardformatter`. Поддерживает автоматическое обнаружение карт, выбор метки раздела и различные методы форматирования.

## ✨ Возможности  
- **Автоматическое обнаружение:** Распознаёт SD-карты, подключённые в режиме реального времени.  
- **Варианты форматирования:** Поддержка «Быстрого форматирования» (Quick) и «Полного форматирования» (Overwrite).  
- **Многоязычность:** Полная поддержка болгарского и английского языков (выбор осуществляется автоматически в зависимости от системных локалей).  
- **Защита:** Предотвращает случайное форматирование системных дисков.  
- **Интеграция:** Отлично работает с KDE (KDialog) и GNOME (Zenity) для системных уведомлений.
